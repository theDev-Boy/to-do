# 🚀 MASTER PROMPT — Premium Glassmorphism To-Do & Task Manager App

---

## ROLE & MISSION

Build a fully-featured, production-ready To-Do & Task Manager Flutter app with premium glassmorphism UI, butter-smooth animations, and deep polish.

## DESIGN PHILOSOPHY

### Visual Language
- **NO gradients** — zero linear-gradient backgrounds. All depth from **glassmorphism only**.
- Every surface: `blur(24–40px)` + `rgba(255,255,255,0.08–0.15)` bg + `1px solid rgba(255,255,255,0.18)` border + shadow
- Background: deep rich **solid dark** color (`#0a0a12` or `#07080f`)
- Accent: electric violet (`#7C3AED`) + amber (`#F59E0B`) for urgency
- Text: white primary, rgba(255,255,255,0.6) secondary

### Navigation Structure
```
Bottom Tab Bar (glass, 5 tabs):
├── 🏠 Home (Today + quick-add)
├── 📋 Tasks (all tasks, filter/sort)
├── ➕ FAB (center, elevated, create task)
├── 📊 Stats (dashboard, calendar)
└── ⚙️ Settings
```

### Core Features Required
- Task CRUD with swipe gestures & animations
- Priority (None/Low/Medium/High/Critical)
- Due dates, categories, tags, subtasks
- Reminders & notifications
- Search & filter
- Dashboard with progress stats
- Calendar view
- Focus/Pomodoro timer mode
- Settings (theme, accent color, notifications)
- 100% offline storage
- No login, no account, no ads
