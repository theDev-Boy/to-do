# ✅ To-Do App

A modern, feature-rich **Task Manager** application built with **Flutter**. Designed to help you organize your daily tasks efficiently with a beautiful glassmorphism UI.

## ✨ Features

### 📋 Core Task Management
- **Create, edit, delete, and complete** tasks with ease
- **Drag-and-drop** reordering of tasks
- **Swipe gestures** for quick actions (complete / delete)
- **Undo** support for accidental deletions
- **Bulk select** mode for batch operations

### 🏷️ Task Properties
- **Priority levels** — None / Low / Medium / High / Critical (color-coded)
- **Due dates & times** with native pickers
- **Categories / Projects** — custom with color + icon
- **Tags** — multi-tag support with autocomplete
- **Subtasks** — nested checklist with progress tracking
- **Recurrence** — daily / weekly / monthly / custom intervals

### 🔔 Reminders & Notifications
- Scheduled reminders per task
- Due-date alerts (15 min before)
- Overdue notifications
- Snooze functionality
- In-app notification center

### 🔍 Search & Filter
- **Full-text search** across titles, descriptions, and tags
- **Filter** by priority, status, category, tags, and due date range
- **Sort** by due date, creation date, priority, title, or completion %
- **Saved filters** for quick access

### 📊 Dashboard & Stats
- **Progress ring** — daily completion tracker
- **Weekly streak** counter
- **Productivity score**
- **Category breakdown** chart
- **7-day completion** bar chart
- **Completion calendar heatmap**

### 📅 Calendar View
- Monthly calendar with task indicators
- Horizontal week strip with swipe navigation
- Day detail panel showing tasks

### ⏱️ Focus / Pomodoro Mode
- 25-minute focus timer with break reminders
- Custom timer durations
- Fullscreen focus mode

### 🕐 History & Activity
- Activity feed with chronological logs
- Edit history per task
- Completion heatmap (GitHub-style)

### ⚙️ Settings
- Dark / Light / System theme
- 8 accent color presets
- Configurable notification preferences
- Data export / import (JSON & CSV)
- App lock (PIN or biometric)

## 🚀 Getting Started

### Prerequisites
- **Flutter SDK** 3.11.1 or higher
- **Dart SDK** 3.11.1 or higher

### Installation

```bash
# Clone the repository
git clone https://github.com/theDev-Boy/to-do.git

# Navigate to the project directory
cd to-do

# Install dependencies
flutter pub get

# Run the app
flutter run
```
